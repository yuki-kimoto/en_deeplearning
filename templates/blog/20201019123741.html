tanh function --activation function Let's write the tanh function in Perl. The tanh function is one of the activation functions . The tanh function is one of the trigonometric functions. Math :: Trig Defined as a function of the module. 
use strict;
use warnings;

# Import tanh function
use Math :: Trig'tanh';

my $ tanh = tanh 3;

# 0.995054753686731
print "$ tanh \ n";
