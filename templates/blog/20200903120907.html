シグモイド関数 - 活性化関数

シグモイド関数をPerlで書いてみます。シグモイド関数は活性化関数の一つです。 
sub sigmoid {
  my ($x) = @_;
  
  my $sigmoid = 1.0 / (1.0 + exp(-$x));
  
  return $sigmoid;
}

my $x = 0.5;
my $sigmoid = sigmoid($x);

# 0.6224593312018545646389
print "$sigmoid\n";
